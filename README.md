# Darwin Threat Intelligence Lookup

A Python cybersecurity project that provides information about common cyber threats and introduces threat intelligence concepts.

## Features

- Looks up common cyber threats
- Provides threat descriptions
- Beginner-friendly Python code
- Introduces threat intelligence concepts

## Supported Threats

- Phishing
- Malware
- Ransomware
- Trojan

## Example

Input:

```
phishing
```

Output:

```
Phishing: Attempts to steal sensitive information.
Created by Darwin Brown
```

## Skills Used

- Python
- User Input
- Conditional Statements
- Threat Intelligence
- Cybersecurity Fundamentals
